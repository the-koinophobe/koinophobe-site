Images for the "How I work" process section.

Drop a screenshot for each step here, then set its `image` in lib/content.ts:
  public/process/rank.png   -> step 01 "Rank for buyers"   -> image: "/process/rank.png"
  public/process/track.png  -> step 02 "Track everything"  -> image: "/process/track.png"
  public/process/prove.png  -> step 03 "Prove it converts" -> image: "/process/prove.png"

Good choices:
  rank   = a SERP result, keyword research, or ranking screenshot
  track  = a GA4 or Google Tag Manager dashboard
  prove  = a Search Console performance graph or a monthly report

Recommended: landscape images ~1200x900 or wider. They fill the right side of each card.
Until added, each card shows a styled placeholder, so the section looks complete either way.
