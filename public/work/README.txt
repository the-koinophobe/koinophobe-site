Drop website screenshots here.

How it works
------------
1. Save a screenshot as a PNG (or JPG) in this folder.
2. Recommended size: 1280 x 800 (16:10). Full-page or above-the-fold both work; the frame crops to the top.
3. Name it with a simple slug, e.g. overthetabletop.png
4. In lib/content.ts, set that project's `image` to "/work/overthetabletop.png"

Suggested filenames for the current projects:
  overthetabletop.png   -> Over The Table Top
  myofascial.png        -> Myofascial Pain Brevard
  tintlordz.png         -> Tintlordz
  hhroofing.png         -> HH Roofing & Repairs
  pbseo.png             -> Palm Bay Marketing SEO

Easy ways to capture a screenshot
---------------------------------
- Chrome: open the site, F12 -> Ctrl+Shift+P -> type "screenshot" -> "Capture full size screenshot".
- Or a free service in your browser: https://s.wordpress.com/mshots/v1/<URL>?w=1280
  (paste the site URL where <URL> is, open it, save the image).

Until an image is added, the card shows a styled "Screenshot coming soon" placeholder, so the page looks fine either way.
